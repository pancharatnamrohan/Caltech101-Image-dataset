# Caltech-101 Image Classification

Mini Project 1 – SHBT 261: Artificial Intelligence in Medicine  
Harvard Medical School

This repository contains the code used for the project:

"Comparative Evaluation of Classical and Deep Learning Methods on Caltech-101 Image Classification"

Models implemented:
- HOG + Linear SVM
- ResNet-18
- EfficientNet-B0

Experiments include:
- Multi-class image classification
- Top-1 and Top-5 accuracy
- Confusion matrix analysis
- Ablation studies (resolution and augmentation)
